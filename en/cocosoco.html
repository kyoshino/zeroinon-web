Telepresence chat app

'cocosoco'


Cocosoco is a video chat app whose name translates to “here and there” in English.
In this app, there is a pointing function, which can be used to point to a space with demonstrative words, which is not possible in a conventional video chat app.
It is designed to make it easier to share a space to encourage dialogue, even online, and to increase the sense of presence as if we were chatting in the same physical place.



Features

Pointing

Online conversation using pointing icons with directives such as "here" and "there”．Pointing icons with face pictures so you can see who is pointing．

Map view

Live streaming as well as geolocation enable spatial understanding from both subjective and objective perspectives.

Zoom

When the live streamer pinches out the screen focusing on the area he/she draws attention to viewer, the live view is also zoomed in．

LIKE

The LIKE function can be used to express a casual interest that does not require pointing.
