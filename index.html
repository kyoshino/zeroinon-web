テレプレゼンスチャットアプリ

「ここそこ」とは

ここそこはスマホやPCを使い，ライブ映像を共有しながら情報伝達を行う簡易テレプレゼンスツールです．
ライブ中継映像の中に指差しアイコンを表示する機能により，通常のオンラインビデオ会議ツールでは使えない，「ここ」や「そこ」「あれ」などの指示語を使った会話が可能です．
指示語を使うことで，あたかもその場に共存しながら自然な会話を楽しむ楽しむことができます．
さらにデジタルならではの特長として中継映像の場所を位置情報として地図上にマッピングして 俯瞰的な場の共有や，遠くの場所をズームによって拡大しながらの会話や，ちょっとした関心・興味を匿名でLIKEできます．



ここそこの特長

指差し

指差しアイコンを使って「ここ」や「そこ」など指示語を使った会話ができます．顔写真のついた指差しアイコンで誰が指差しをしているかわかります．

地図表示

ライブ中継映像と位置情報を同時に見ることで，主観視点と客観視点で空間把握が可能になります.

ズーム

中継者が画面をピンチアウトするとライブ映像もズームされます．中継者が遠くの場所に注目してもらいたい時などズームしながら指差し会話できます．

LIKE

画面を長押しするとハートマークのアニメーションが表示されます．指差しするほどでもない時にオンラインの参加者に軽くエモーションを伝えましょう．

ここそこの使い方マニュアル.


ここそこはブラウザアプリです．使用の際にアプリケーションを事前にインストールせずブラウザを立ち上げるだけで利用可能です．対応機種とブラウザ
  • Macintosh PC: google chrome firefox, safari(最新版である必要はありませんがカメラ，マイクを認識するバージョンが必要です）
  • Windows PC: google chrome, fireforx, Microsoft edge(最新版である必要はありませんがカメラ，マイクを認識するバージョンが必要です）
  • iPhone, iPad: safari(最新版である必要はありませんがカメラ，マイクを認識するバージョンが必要です）

    • * マイク付きヘッドホンを使用するとマイクと発話者の口元までの距離が近いので雑音を拾いにくく比較的クリアな音質で会話できます．また周囲への音漏れも気にすることなく音も聞きやすく スピーカーの音をマイクが拾ってハウリングになるリスクも抑えられます．
    * ここそこで良いテレプレゼンスチャット体験が得られるかどうかはネットワークの回線速度に依存します．自分のカメラで中継をする予定のある人は，約2Mbps以上の通信速度（アップロード側）が必要です．携帯電話のデータ通信サービスで通信制限がかかると中継映像の更新が遅れたり，音声が途切れやすくなります． また中継せずに参加する人も中継映像を受け取り音声通話をするために約2Mbps以上（ダウンロード側）の通信速度が必要です． ご自身の通信環境を確認したい場合は，ブラウザで'test my speed'と検索し速度テストを行うか， こちらをクリックして速度テストを実施して環境を事前にご確認することをお勧めします.
